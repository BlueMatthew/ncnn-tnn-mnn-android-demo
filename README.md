# ncnn & tnn & mnn 的安卓demo工程

为方便平时使用，这里把ncnn、tnn、mnn都塞进了安卓工程里，有实时摄像头处理的工程和从相册选图单次推理的工程。

平时开发要用到的时候直接复制一个工程就行了。三个框架全塞了进去也不用再愁怎么配，大力出奇迹。
